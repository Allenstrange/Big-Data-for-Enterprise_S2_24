  Assessment repository for **DAS7001 — Big Data for Enterprise**, part of the MSc Data Science programme (Semester 2, 2024–25).

  This module examines how organisations harness Big Data to gain competitive advantage, covering NoSQL database design, enterprise data architecture, and scalable data solutions.

  ---

  ## Technologies & Tools

  - **MongoDB** — NoSQL document database design and querying
  - **PyMongo** — Python interface for MongoDB operations
  - **Python** — pandas, data processing and transformation
  - **Jupyter Notebooks** — interactive analysis and documentation
  - **Entity Relationship Diagrams (ERD)** — database schema design

  ## Module Overview

  Big Data for Enterprise explores how businesses leverage data assets at scale. Key topics include:

  - NoSQL vs relational database trade-offs in enterprise contexts
  - MongoDB schema design patterns and best practices
  - Big Data governance, security, and compliance
  - Enterprise data lakes and warehouse architectures
  - Business intelligence and data-driven decision making
